# Subset ShapeMap objects

Subsetting ShapeMap objects essentially passes the data to sf. See
[sf](https://r-spatial.github.io/sf/reference/sf.html)

## Usage

``` r
# S3 method for class 'ShapeMap'
x[...]

# S3 method for class 'ShapeMap'
x[...] <- value
```

## Arguments

- x:

  object of class `ShapeMap` passed to `sf[]`

- ...:

  other parameters passed to `sf[] <- `

- value:

  value to be passed to `sf[] <- `
